Импорт проектов из Excel

Один Excel — несколько проектов. Одна вкладка = один проект (ключи + частотность + домен на вкладке). Колонки распознаются по синонимам в любом порядке, KD игнорируется, домен находится автоматически (http→https), пустые и сводные листы пропускаются. Проект ищется по домену: нет — создаётся, есть — обновляется. Результат по каждой вкладке — на странице «Логи».
