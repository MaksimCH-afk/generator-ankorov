Пакетная загрузка частоток

Загружайте частотки сразу для нескольких проектов. Поддерживаются два варианта:

  • Один Excel с вкладками — каждая вкладка = частотка одного проекта. Имя вкладки сопоставляется с проектом по его домену или бренду (напр. вкладка betalice → проект с брендом «betalice» или доменом betalice.com). В одном файле может быть сколько угодно проектов.
  • Несколько отдельных файлов (CSV или Excel) — один файл = один проект; тогда сопоставление идёт по имени файла (домен или бренд в названии).

Формат данных: в первой колонке — ключ, во второй — частотность (заголовок вроде Keyword | Volume распознаётся и пропускается). Загрузка заменяет текущую частотку проекта. Подробный результат по каждой вкладке/файлу — на странице «Логи».

ℹ️ Справка: с чего начать

Эта страница — список ваших проектов (доменов). Один проект = один домен, для которого мы рассчитаем план размещения ссылок и выгрузим Excel. Начните с кнопки «Создать проект»: укажите URL домена, выберите язык статьи (или «Не указывать в выгрузке», если язык в финальном файле не нужен) и бренд (он подставляется в анкоры внутренних страниц).

После создания проекта вы попадёте на его страницу, где нужно загрузить частотку (файл «ключ; частотность»), выбрать стратегию распределения анкоров и задать объёмы ссылок. Если проектов много, удобнее залить все частотки сразу через «Пакетную загрузку» — просто назовите файлы по домену или бренду проекта.

Колонка «Готовность» подсказывает, какие проекты уже можно генерировать. Когда всё настроено, переходите в раздел «Генерация», отмечайте нужные проекты и скачивайте результат в ZIP или отдельными Excel-файлами.
