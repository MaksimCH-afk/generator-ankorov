Импорт проектов из Excel

Загрузите один Excel — и сразу создадутся (или обновятся) несколько проектов. Одна вкладка = один проект, и на вкладке есть всё: ключи, частотность и домен.

  • Колонки распознаются по названиям-синонимам (Keyword/Ключ/Запрос, Volume/Частотность и т.п.) и могут стоять в любом порядке. Колонка KD / Keyword Difficulty игнорируется.
  • Домен ищется автоматически — в заголовке, отдельной колонке или в каждой строке. Адрес вида http://site.at/ приводится к https://site.at/.
  • Проект ищется по домену: если такого нет — создаётся новый, если есть — обновляется его частотка.
  • Пустые вкладки и сводные/дашборд-листы (без ключей и доменов) просто пропускаются.

Можно загрузить сразу несколько файлов. Подробный результат по каждой вкладке — на странице «Логи».

ℹ️ Справка: с чего начать

Эта страница — список ваших проектов (доменов). Один проект = один домен, для которого мы рассчитаем план размещения ссылок и выгрузим Excel. Начните с кнопки «Создать проект»: укажите URL домена, выберите язык статьи (или «Не указывать в выгрузке», если язык в финальном файле не нужен) и бренд (он подставляется в анкоры внутренних страниц).

Можно действовать двумя путями. Вручную: создать проект, затем на его странице загрузить частотку, выбрать стратегию и задать объёмы. Массово: через «Импорт проектов из Excel» — один файл, где каждая вкладка содержит ключи, частотность и домен; проекты создадутся автоматически (а дальше останется только выбрать стратегию и объёмы).

Колонка «Готовность» подсказывает, какие проекты уже можно генерировать. Когда всё настроено, переходите в раздел «Генерация», отмечайте нужные проекты и скачивайте результат в ZIP или отдельными Excel-файлами.
