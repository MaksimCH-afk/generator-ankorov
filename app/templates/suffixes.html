ℹ️ Справка: внутренние страницы

Здесь хранится общий справочник анкоров для типовых внутренних страниц (app, login, bonus, withdraw, deposit и т.д.). Смысл страниц одинаков для всех проектов, но текст анкора зависит от языка. Для каждого типа страницы задаётся суффикс на каждом языке локализации (en, de, pl, tr, pt-br).

При генерации анкор внутренней страницы собирается как {бренд} {суффикс}. Например, для бренда «betalice», страницы «withdraw» и немецкого языка суффикс «auszahlung» даст анкор betalice auszahlung. В блоке «Внутренние страницы проекта» сверху выберите проект, задайте язык анкоров и пропишите соответствие «тип страницы → URL-путь» (например, withdraw → /boni/).

Чтобы добавить новый тип страницы — впишите его название и заполните суффиксы по языкам. Пустые поля просто не сохраняются. Этот справочник — общий, поэтому изменения отразятся во всех проектах при следующей генерации.
