ℹ️ Справка: настройка проекта

Страница собрана по шагам — двигайтесь сверху вниз по пронумерованным блокам. Сначала задайте основные параметры (URL, язык статьи, бренд). Затем (1) загрузите частотку — файл с колонками «ключ» и «частотность»; ключи сами разложатся по ролям. (2) Выберите стратегию — в её названии показаны вид безанкора и проценты по ролям; тип кампании (прогоны или крауд+сабмиты) определяется именно стратегией. (3) Задайте объём — общее число ссылок на URL.

Когда все три шага отмечены галочками, блок (4) Генерация станет «готов» — открывайте «Превью результата» для проверки и «К генерации» для выгрузки в Excel. Каждый блок сохраняется своей кнопкой независимо, поэтому можно менять что-то одно, не трогая остальное.

Анкоры внутренних страниц (app, login, bonus и т.п.) вынесены в раздел «Внутренние страницы» — там для каждого проекта задаётся язык и соответствие «тип страницы → URL-путь».
