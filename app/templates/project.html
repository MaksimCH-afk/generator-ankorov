ℹ️ Справка: настройка проекта

Здесь собирается всё для расчёта плана по одному домену. Порядок простой и показан в блоке «Что нужно сделать» сверху: (1) загрузите частотку внизу страницы — файл с колонками «keyword» и «frequency»; ключи автоматически разложатся по ролям (основной 1/2, добавочный 1/2…) по убыванию частотности. (2) Выберите стратегию — это профиль процентов (сколько ссылок идёт в безанкор и на каждую роль). (3) Задайте объёмы: «Прогоны» — общий профиль анкоров, «Крауд+сабмиты» — отдельный объём, который весь уходит в безанкор.

Блок «Внутренние страницы» нужен, если кроме главной вы продвигаете типовые страницы (app, login, bonus…). Выберите язык анкоров и добавьте пары «тип страницы → URL-путь»; анкор соберётся как {бренд} {суффикс} (суффиксы редактируются в разделе «Внутренние страницы»). Поле «Перераспределение» — для тонкой настройки: если для какой-то роли нет ключа, по умолчанию её процент уходит в безанкор, но здесь можно перенаправить его на конкретную роль.

Когда все шаги отмечены галочками, нажмите «Превью результата», чтобы увидеть готовые строки по вкладкам, и затем «К генерации» — для скачивания Excel. Не забудьте «Сохранить проект» после изменения параметров.
